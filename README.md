# OnlySQ AI Chrome Extension

Интеллектуальный ассистент для браузера Google Chrome, интегрированный прямо в боковую панель (Side Panel). Расширение выполнено в премиальном бруталистичном (терминальном) дизайне и предоставляет мощный набор инструментов на базе API OnlySQ, включая полноценный агентный режим взаимодействия с веб-страницами.

---

## Ключевые возможности

*   **Интерактивный чат-интерфейс**:
    *   Премиальный бруталистичный (терминальный) дизайн с темным интерфейсом, кастомными шрифтами, контрастными рамками и плавными микро-анимациями.
    *   Полноценное форматирование сообщений с поддержкой Markdown (заголовки, списки, выделение жирным и курсивом, inline-код и блоки кода с авто-экранированием).
    *   Быстрые действия с сообщениями: копирование чистого текста в буфер обмена, повторная генерация ответов.
    *   Сворачиваемые блоки системных рассуждений (`<think>`) для удобного чтения.

*   **Агентный режим (Multi-step Agent)**:
    *   Автономный пошаговый цикл выполнения задач пользователя (до 5 шагов).
    *   Анализ содержимого открытых страниц и автоматическое выполнение действий.
    *   Поддерживаемые команды в ответах ИИ:
        *   `[ACTION: OPEN_TAB, url: "URL"]` — фоновое открытие веб-страницы.
        *   `[ACTION: CLICK, text: "ТЕКСТ"]` — клик по кнопке или ссылке с указанным текстом.
        *   `[ACTION: SCROLL, direction: "down" | "up"]` — прокрутка страницы.
        *   `[ACTION: WAIT, seconds: N]` — пауза для загрузки динамического содержимого.
        *   `[ACTION: CLOSE_TAB]` — закрытие вкладки, созданной агентом.
    *   Параллельный опрос вкладок при сборе контекста с защитой от зависаний (таймаут 1.5 секунды на вкладку).

*   **Гибкий контекст запросов**:
    *   **Нет (None)**: Стандартный режим общения с моделью.
    *   **Вкладка (Page)**: Использование текстового содержимого текущей активной вкладки в качестве контекста.
    *   **Все вкладки (All Tabs)**: Анализ текстов со всех открытых вкладок текущего окна браузера.
    *   **Зрение (Vision)**: Быстрый захват скриншота активного окна и его отправка в мультимодальную модель.

*   **Настройка уровня рассуждений (Reasoning Effort)**:
    *   Выбор уровня усилий для рассуждений модели напрямую из шапки чата: `Minimal`, `Low`, `Medium`, `High`.
    *   Полная совместимость параметров рассуждения с моделями семейства DeepSeek R1 и аналогичными через API-шлюз.

*   **Адаптивность**:
    *   При уменьшении ширины боковой панели до `500px` и менее, сайдбар истории чатов разворачивается на всю ширину экрана для сохранения читаемости и предотвращения сжатия элементов.

## Структура проекта

Код расширения структурирован по функциональным каталогам для обеспечения чистоты и удобства поддержки:

```text
onlysq-ai-chrome-extension/
├── css/
│   └── styles.css              # Стили интерфейса, переменные темы, анимации и бруталистичная сетка
├── icons/
│   ├── icon16.png              # Иконка расширения (16x16)
│   ├── icon48.png              # Иконка расширения (48x48)
│   └── icon128.png             # Иконка расширения (128x128)
├── js/
│   ├── background.js           # Фоновый сервис-воркер (захват скриншотов, управление вкладками)
│   ├── content.js              # Скрипт для извлечения текстового содержимого из страниц
│   ├── options.js              # Логика настроек и проверки API-ключа
│   ├── popup.js                # Скрипт для всплывающего окна (Popup)
│   └── sidepanel.js            # Логика чата, стриминга ответов, Markdown и агентного цикла
├── ui/
│   ├── options.html            # Страница авторизации и настроек (sq-... ключ, ссылки)
│   ├── popup.html              # Всплывающее меню расширения
│   └── sidepanel.html          # Основной интерфейс боковой панели чата
├── manifest.json               # Конфигурационный файл Chrome Extension (Manifest V3)
├── LICENSE                     # Лицензионное соглашение
└── README.md                   # Документация проекта (этот файл)
```

## Установка и запуск

1.  **Скачайте исходный код** проекта (склонируйте репозиторий или скачайте ZIP-архив и распакуйте его).
2.  Откройте браузер Google Chrome и перейдите по адресу `chrome://extensions/`.
3.  В правом верхнем углу включите переключатель **Режим разработчика** (Developer mode).
4.  В левом верхнем углу нажмите кнопку **Загрузить распакованное расширение** (Load unpacked).
5.  Выберите корневую папку проекта `onlysq-ai-chrome-extension` (ту, в которой находится файл `manifest.json`).
6.  Закрепите расширение на панели инструментов. Для открытия нажмите на иконку ассистента или откройте Side Panel вручную через меню браузера.

## Настройка API-ключа

1.  Перейдите на страницу настроек расширения (нажмите шестеренку в левом нижнем углу боковой панели или кликните правой кнопкой мыши по иконке расширения и выберите **Параметры**).
2.  Введите ваш секретный ключ API OnlySQ (формата `sq-...`), который можно получить в личном кабинете на сайте [my.onlysq.ru](https://my.onlysq.ru).
3.  Нажмите кнопку **Сохранить и проверить** для валидации токена и сохранения настроек.
4.  Внизу страницы настроек также доступны быстрые ссылки на сайт проекта OnlySQ и официальный репозиторий GitHub.
